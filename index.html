What is arrow

  • KΛTEGORY + funKTionale = arrow
    • The two most relevant FP libraries for Kotlin in 2017
  • Functional companion to Kotlin's Standard Library

Multiple modules

Examples

Option is deprecated

Option will be deleted soon as it promotes the wrong message of using a slower and memory unfriendly abstraction when the lang provides a better one

Either

Either is used to short-circuit a computation upon the first error

By convention, the right side of an Either is used to hold successful values

Arrow fx

A next-generation Typed FP Effects Library that makes tracked effectful programming first class in Kotlin, and is build on top of Kotlin’s suspend system

Arrow fx

Notice that raceN(...) returns Either

Arrow optics

Optics are essentially abstractions to update immutable data structures in an elegant way

Lens

A Lens is an optic that can focus into a structure and get, modify, or set its focus (target)

Lenses can be generated for a data class by the @optics annotation

Λrrow Meta is a meta-programming library that cooperates with the Kotlin compiler in all its phases, bringing its full power to the community
